# Ferramentas

- account-checker
    - (✅) Verifica se a conta tem email de recuperacao cadastrada
    - (✅) Verifica se a senha esta correta
    - (✅) Verifica o Rated Limited, evitar spam
    - (✅) Troca a senha automaticamente

- cookies-checker
    - (✅) Verifica se o cookie ".ROBLOSECURITY" funciona
